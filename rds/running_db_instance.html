Run an RDS DBInstance.

Core Concepts

Amazon RDS DBInstance

A database instance is an EC2 instance running the database. Database is stored on an EBS-like SAN storage but is is ephemeral unless you save a snapshot at termination.

Launching a DB Instance

1. In the EC2Dream System Admin click on testdb under RDS and the launch profile tab will open.



2. Specify the DBName, Availability Zone, Master user name, Master password and DB Parameter Group Name.



3. Launch the DB Instance by pressing the Launch button .



4. To access the MySQL Administrator, enter the Master user password and click on the icon in the Server tab.



NOTE: This assumes the MySQL GUI Tools are installed on your client.

5. To modify the DBInstance press on the icon.



6. Enter the values to modify and press the modify button.

7. To delete the DB instance click on the icon .



8. Either click Skip Final Snapshot or specify the final snapshot id and press the Delete button.
