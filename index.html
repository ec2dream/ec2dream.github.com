EC2Dream Documentation

EC2Dream is a graphic user interface that provides a 'single pane of glass' to do agile devops primarily on cloud servers by using:
-Rapid agile iterative development of chef cookbooks, puppet modules or ansible playbooks to build servers.
-Test Kitchen to deploy and test them.
-Cloud Formation or Terraform using cfndsl Ruby or Troposphere DSL for easy orchestration.
-Amazon AWS, Azure, Google Compute Engine, Softlayer, Openstack, Local and Hosted servers.
-One click ssh, rdp, and sftp access to servers.
