EC2Dream Fogviz Documentation

EC2Dream Fogviz is visual cloud computing admin for the Fog ruby cloud services library and combines Fog and Ruby into an open source devops platform that is particularly helpful when writing chef or puppet scripts. Currently it supports:
      iterative development of chef cookbooks and puppet modules.
      Amazon AWS including the ability to list most entities.
      Amazon compatible clouds:  Eucalyptus, CloudStack.
      Openstack Clouds:  Rackspace Cloud Servers and HP Cloud.
      Cloud Foundry and even Local Servers.

It uses Chef Solo or Puppet Apply to install software and applications via "one click" .
